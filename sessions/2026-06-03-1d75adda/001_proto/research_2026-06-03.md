# Research: Update Protobuf Interfaces for Feedback and Context

**Date**: 2026-06-03

## 1. Executive Summary
The goal is to enhance the `AiService` (specifically the `DecideBrowserAction` RPC) by providing more context about the current page state and feedback from the previous action. This involves updating `ai.proto` with new fields and structured data types for DOM elements (links, buttons, forms).

## 2. Technical Context
- **Existing Implementation**: `proto/ai/v1/ai.proto` defines `DecideBrowserActionRequest` with basic fields: `url`, `page_source`, and `current_goal`.
- **Generation Process**: `Taskfile.yml` defines a `proto` task that runs `buf generate` for Go and TS, and `protoc` for Python.
- **Affected Components**: 
    - `AiService` in Python (`ai/src/server.py`)
    - Go Orchestrator (`internal/orchestrator/orchestrator.go`)
    - UI generated stubs (`ui/src/gen/ai/v1/ai_pb.ts`)

## 3. Findings & Analysis
- **DecideBrowserActionRequest**: Needs expansion to include feedback (`last_action_success`, `last_action_error`) and better page context (`current_url`, `page_title`, and structured elements).
- **Structured Elements**: Currently, the AI service parses the raw `page_source` (truncated to 5000 chars in `server.py:291`). Providing structured lists of interactive elements will allow for more precise decisions and potentially reduce token usage or improve accuracy.
- **Python Stubs**: Python stubs are generated into `ai/src/proto`. The `server.py` adds this directory to `sys.path`.

## 4. Technical Constraints
- **Protobuf Field Numbers**: New fields should be added with sequential numbers starting from 4 in `DecideBrowserActionRequest`.
- **Generation Tools**: `buf` and `grpcio-tools` (python) must be available. The `Taskfile.yml` points to specific paths for these tools.

## 5. Architecture Documentation
- **Service Pattern**: The project uses ConnectRPC for Go and TS, and standard gRPC for Python.
- **Proto Location**: All proto definitions are centralized in the `proto/` directory.
- **Generated Code Location**: 
    - Go: `internal/gen/`
    - TS: `ui/src/gen/`
    - Python: `ai/src/proto/`
